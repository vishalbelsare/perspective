Returns the column names of this [`Table`] in "natural" order (the ordering
implied by the input format).

<div class="javascript">

# JavaScript Examples

```javascript
const columns = await table.columns();
```

</div>
<div class="python">

# Python Examples

```python
columns = table.columns()
```

</div>
<div class="rust">
